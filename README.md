# 象棋麻將

自創桌遊，結合象棋棋子與麻將玩法。

🎮 **立即遊玩**
- [單機熱座版](https://yaojing277.github.io/chess-mahjong/)
- [多人連線版](https://yaojing277.github.io/chess-mahjong/multi.html)（Firebase 即時同步，最多 4 人）

---

## 牌堆組成

共 32 隻棋子，紅黑各 16：

| 顏色 | 棋子 |
|------|------|
| 紅 | 帥×1、仕×2、相×2、俥×2、傌×2、炮×2、兵×5 |
| 黑 | 將×1、士×2、象×2、車×2、馬×2、包×2、卒×5 |

---

## 遊戲流程

1. 4 人各發 4 隻起手牌，由莊家先行，每局莊家輪替
2. **每回合**：摸 1 隻（手中 5 隻）→ 判斷胡牌 → 未胡則打出 1 隻（回到 4 隻）
3. **棄牌後**：自動檢查其他玩家是否能以該牌胡牌（放砲）
4. 無人胡牌 → 下家可選擇**吃牌**（取得棄牌）或**摸牌**（跳過吃牌直接抽）

---

## 胡牌條件（手中 5 隻）

| 主牌（3 隻） | 顏色 | 搭配 |
|-------------|------|------|
| 帥 + 仕 + 相 | 全紅 | + 對子 |
| 將 + 士 + 象 | 全黑 | + 對子 |
| 俥 + 傌 + 炮 | 全紅 | + 對子 |
| 車 + 馬 + 包 | 全黑 | + 對子 |
| 3 隻紅兵 | 全紅 | + 對子 |
| 3 隻黑卒 | 全黑 | + 對子 |
| 5 隻紅兵 | 全紅 | — |
| 5 隻黑卒 | 全黑 | — |

### 對子規則

- 兩張**同色且同名**才能構成對子
- 唯一例外：**帥（紅）＋ 將（黑）**可互相成對

## 技術架構（多人版）

- 純前端 HTML + JavaScript，單一 `.html` 檔案
- Firebase Realtime Database 即時同步，無需後端
- 以 `localStorage` UUID 識別玩家，免登入
- 支援手機與桌機
